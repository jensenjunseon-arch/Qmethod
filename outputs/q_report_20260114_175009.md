## 1. 연구 개요

### 📌 연구 주제
**직장인이 조직 내 가치 불일치를 경험하는 다양한 유형 탐색**

| 항목 | 내용 |
|------|------|
| 연구 질문 | 직장인들이 조직 내 가치 불일치를 어떻게 경험하고 이러한 경험은 어떤 유형으로 구분될 수 있는가? |
| 대상 집단 | 다양한 산업과 직종에 종사하는 직장인 |
| 연구 맥락 | 조직 내에서 개인의 가치와 조직의 가치가 불일치하는 상황을 겪는 직장인들의 경험을 분석하여 유형화 |
| 핵심 키워드 | 조직가치, 불일치, 직장인, 유형화, 경험 |
